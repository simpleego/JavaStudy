# 💻 컴퓨터 기초 학습 사이트 정리
> 컴퓨터 기초부터 프로그래밍 입문까지 학습할 수 있는 온라인 사이트들을 정리해 드립니다. 대부분 무료 강좌를 제공하며, 난이도별/주제별로 다양하게 선택할 수 있습니다.

### 1. 일반 컴퓨터 활용 및 IT 기초 강좌

| 사이트 이름 | 특징 | 주요 내용 |
| :--- | :--- | :--- |
| **K-MOOC** (한국형 온라인 공개 강좌) | 국내 대학 및 기관의 **무료** 온라인 공개 강좌 제공 | 컴퓨터 네트워크, UX/UI 디자인 입문, 인공지능 등 다양한 IT 및 전공 분야 강좌 () |
| **한컴캠퍼스** | 온라인 강좌 제공 | 한컴오피스(한글), 컴퓨터 활용 등 |
| **에듀윌 국비교육원** | 사무행정 입문 과정 등 제공 (온라인/오프라인) | 왕초보 컴퓨터 기초 (윈도우, 인터넷, 한글 등) |
| **길벗 (특별체험판 PDF)** | 컴퓨터 기초 지식 다지기 자료 (PDF) | 컴퓨터 자기소개, 마우스/키보드, 파일/폴더 관리 등 |

### 2. 코딩 및 프로그래밍 기초 강좌 (컴퓨터 원리 이해에 도움)

| 사이트 이름 | 특징 | 주요 내용 |
| :--- | :--- | :--- |
| **생활코딩** (Open Tutorials) | 일반인에게 프로그래밍을 알려주는 **비영리 무료** 교육 프로젝트 | 웹서비스(HTML, CSS, JavaScript) 중심, 프로그래밍 언어, Git 등 기초부터 심화까지 () |
| **인프런** | 온라인 학습, 지식 공유 중개 플랫폼 (무료/유료 강좌 혼재) | 개발, 프로그래밍, MS-Office, AI 활용(AX), SQL 등 IT 전반의 실전 지식 |
| **구름 EDU** | 모두를 위한 맞춤형 IT 교육 플랫폼 | 기초부터 실무 프로그래밍 교육, 클라우드 기반 개발 환경(구름 IDE) 제공 |
| **code.org** | 구글, MS 등이 지원하는 **비영리** 교육기관 | 처음 코딩을 접할 때 좋은 기초 사이트, 다양한 무료 코딩 사이트 링크 |
| **소프트웨어야 놀자** / **엔트리** | 우리나라의 대표적인 코딩 교육 사이트 (과학기술정보통신부 지원) | 코딩뿐 아니라 컴퓨터 지식, 블록형/텍스트형(엔트리 파이썬) 코딩 학습 |
| **코드잇** | 그래픽 기반의 몰입형 학습 경험 제공 (유료 멤버십, 일부 무료) | 코딩 기초, 파이썬, 웹 개발, 알고리즘 등 |
