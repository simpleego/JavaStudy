# Hash와 Map
> 자바에서 **Hash(해시)**는 데이터를 빠르고 효율적으로 저장하고 검색하기 위한 기법입니다.
> 해시 함수가 입력값을 **고유한 정수 값(해시 코드)**으로 변환하고,
> 이를 기반으로 해시 테이블(Hash Table) 같은 자료구조에서 데이터를 관리합니다.  

---

## 🔑 자바에서 Hash란?

- **정의**: 임의 길이의 데이터를 **고정된 길이의 해시 코드(hash code)**로 매핑하는 알고리즘.  
- **목적**: 데이터를 빠르게 찾거나 저장하기 위해 사용.  
- **핵심 개념**:  
  - **Hash 함수**: 입력값을 해시 코드로 변환하는 함수.  
  - **Hash Code**: 객체를 식별하기 위한 정수 값.  
  - **Hash Table**: 해시 코드를 기반으로 데이터를 저장하는 자료구조.  

---

## ⚙️ 특징

- **빠른 검색 성능**: 평균적으로 O(1) 시간에 접근 가능.  
- **Key-Value 구조**: `HashMap`, `Hashtable` 등에서 Key를 해시 코드로 변환해 Value를 찾음.  
- **중복 처리**: Key는 중복 불가. 같은 Key를 넣으면 기존 값이 덮어쓰기 됨.  
- **충돌(Collision)**: 서로 다른 데이터가 같은 해시 코드로 변환될 수 있음. 이를 해결하기 위해 **체이닝(Chaining)**, **오픈 어드레싱(Open Addressing)** 같은 기법을 사용.

## 💻 예제 코드

### HashMap 사용 예제
```java
import java.util.HashMap;

public class HashExample {
    public static void main(String[] args) {
        // HashMap 생성
        HashMap<String, Integer> scores = new HashMap<>();

        // 데이터 추가
        scores.put("국어", 90);
        scores.put("수학", 85);
        scores.put("영어", 95);

        // 값 가져오기
        System.out.println("수학 점수: " + scores.get("수학"));

        // Key 중복 → 값 덮어쓰기
        scores.put("영어", 100);
        System.out.println("영어 점수(갱신 후): " + scores.get("영어"));

        // 전체 출력
        System.out.println("전체 성적: " + scores);
    }
}
```

**실행 결과 예시:**
```
수학 점수: 85
영어 점수(갱신 후): 100
전체 성적: {국어=90, 수학=85, 영어=100}
```

## ⚠️ 주의할 점
- **좋은 해시 함수**는 입력값이 비슷해도 해시 코드가 고르게 분포되어야 함.  
- 해시 기반 자료구조는 **검색과 삽입에 강점**이 있지만, 순서 유지가 필요할 때는 적합하지 않음.  
- 보안 목적의 해시(예: SHA-256)와 데이터 구조 목적의 해시는 구분해야 함.
