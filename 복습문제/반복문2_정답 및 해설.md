# ✅ 1. 3의 배수 / 5의 배수 / FizzBuzz

### ✅ 정답 코드
```java
public class Q1 {
    public static void main(String[] args) {
        for (int i = 1; i <= 100; i++) {
            if (i % 15 == 0) System.out.println("FizzBuzz");
            else if (i % 3 == 0) System.out.println("Fizz");
            else if (i % 5 == 0) System.out.println("Buzz");
            else System.out.println(i);
        }
    }
}
```

### ✅ 해설
- 3과 5의 공배수는 15 → 가장 먼저 체크해야 함  
- 조건 순서가 핵심

# ✅ 2. 1부터 N까지 합, 단 50 초과 시 종료

### ✅ 정답 코드
```java
import java.util.Scanner;

public class Q2 {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        int N = sc.nextInt();
        int sum = 0;

        for (int i = 1; i <= N; i++) {
            sum += i;
            if (sum > 50) break;
        }

        System.out.println("합: " + sum);
    }
}
```

### ✅ 해설
- 누적합이 50을 넘으면 `break`로 반복 종료

# ✅ 4. 구구단 전체 출력 (2~5단 먼저)

### ✅ 정답 코드
```java
public class Q4 {
    public static void main(String[] args) {

        // 2~5단 출력
        for (int dan = 2; dan <= 5; dan++) {
            for (int i = 1; i <= 9; i++) {
                System.out.printf("%d x %d = %d\n", dan, i, dan * i);
            }
            System.out.println();
        }

        // 6~9단 출력
        for (int dan = 6; dan <= 9; dan++) {
            for (int i = 1; i <= 9; i++) {
                System.out.printf("%d x %d = %d\n", dan, i, dan * i);
            }
            System.out.println();
        }
    }
}
```

### ✅ 해설
- 이중 반복문  
- 요구사항: 2~5단 먼저 출력 → 두 개의 반복문 블록으로 분리

# ✅ 7. 입력한 숫자 중 가장 큰 값 찾기

### ✅ 정답 코드
```java
import java.util.Scanner;

public class Q7 {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        int max = Integer.MIN_VALUE;

        while (true) {
            int num = sc.nextInt();
            if (num == 0) break;
            if (num > max) max = num;
        }

        System.out.println("최댓값: " + max);
    }
}
```

### ✅ 해설
- 0 입력 시 종료  
- 입력값이 max보다 크면 갱신

# ✅ 8. 10~50 사이 숫자 개수 세기

### ✅ 정답 코드
```java
import java.util.Scanner;

public class Q8 {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        int count = 0;

        while (true) {
            int num = sc.nextInt();
            if (num == 0) break;
            if (num >= 10 && num <= 50) count++;
        }

        System.out.println("개수: " + count);
    }
}
```

### ✅ 해설
- 조건 범위 체크: `num >= 10 && num <= 50`

# ✅ 10. 숫자 맞히기 게임

### ✅ 정답 코드
```java
import java.util.*;

public class Q10 {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        Random r = new Random();

        int answer = r.nextInt(10) + 1;
        int guess = 0;

        while (guess != answer) {
            guess = sc.nextInt();

            if (guess == answer) System.out.println("정답!");
            else System.out.println("다시 시도");
        }
    }
}
```

### ✅ 해설
- 난수: 1~10  
- 정답이 나올 때까지 반복
