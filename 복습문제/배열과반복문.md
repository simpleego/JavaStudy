# ✅ **기초 배열 문제 10문제**

### **1. 배열에 난수 채우기**
길이가 10인 int 배열을 만들고, 1~100 사이 난수로 채운 뒤 모든 값을 출력하시오.

### **2. 배열의 최댓값·최솟값 찾기**
난수로 채운 배열에서 최댓값과 최솟값을 찾아 출력하시오.

### **3. 배열의 합과 평균 구하기**
난수로 채운 배열의 총합과 평균을 출력하시오.

### **4. 짝수·홀수 개수 세기**
배열에 저장된 값 중 짝수와 홀수의 개수를 각각 출력하시오.

### **5. 특정 값 찾기 (Linear Search)**
배열에 난수로 값을 채운 뒤, 사용자에게 숫자를 입력받아 배열 안에 존재하는지 출력하시오.

### **6. 배열 역순 출력**
배열을 생성하고 난수로 채운 뒤, 원본 순서와 역순을 모두 출력하시오.

### **7. 배열에서 특정 범위 값 출력**
배열에서 50 이상인 값만 출력하시오.

### **8. 배열 요소 교환**
배열의 첫 번째 값과 마지막 값을 서로 교환한 뒤 출력하시오.

### **9. 배열에서 중복 값 개수 세기**
배열을 난수로 채운 뒤, 사용자에게 숫자 하나를 입력받아 배열에서 몇 번 등장하는지 출력하시오.

### **10. 배열 정렬(오름차순)**
배열을 난수로 채운 뒤, 간단한 정렬 알고리즘(버블 정렬 등)을 사용해 오름차순으로 정렬하시오.

# ✅ **고급 배열 문제 10문제**

### **1. 로또 번호 생성기**
1~45 사이 난수 6개를 생성하여 배열에 저장하고, 중복 없이 출력하시오.

### **2. 학생 점수 분석 프로그램**
학생 10명의 점수를 배열에 난수로 저장하고  
- 평균  
- 최고점  
- 최저점  
- 평균 이상 학생 수  
를 출력하시오.

### **3. 난수 기반 빈도수 분석**
1~10 사이 난수를 100번 생성하여 배열에 저장하고,  
각 숫자가 몇 번 등장했는지 빈도수를 출력하시오.

### **4. 배열 섞기(Shuffle)**
1~10까지 값을 가진 배열을 만든 뒤, 난수를 이용해 배열을 무작위로 섞으시오.

### **5. 중복 제거 배열 만들기**
1~20 사이 난수를 10개 생성해 배열에 저장하되,  
중복이 나오면 다시 뽑아서 **중복 없는 배열**을 완성하시오.

### **6. 2차원 배열 난수 채우기**
3×3 배열을 만들고 1~9 사이 난수로 채운 뒤 출력하시오.

### **7. 빙고판 생성**
5×5 배열을 만들고 1~25까지 숫자를 넣은 뒤, 난수로 섞어 빙고판을 완성하시오.

### **8. 배열 회전**
배열을 오른쪽으로 한 칸씩 회전시키는 프로그램을 작성하시오.  
예: {1,2,3,4,5} → {5,1,2,3,4}

### **9. 난수 기반 점수 등급 분류**
학생 20명의 점수를 난수로 생성하고  
A(90~100), B(80~89), C(70~79), D(60~69), F(0~59)  
각 등급별 학생 수를 출력하시오.

### **10. 배열 두 개 합치기**
두 개의 배열을 난수로 생성한 뒤,  
두 배열을 하나의 배열로 합쳐 출력하시오.
