# ✅ 1. 1부터 N까지 소수 출력

### ✅ **정답 코드**
```java
import java.util.Scanner;

public class Q1 {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        int N = sc.nextInt();

        for (int i = 2; i <= N; i++) {
            boolean isPrime = true;

            for (int j = 2; j <= i - 1; j++) {
                if (i % j == 0) {
                    isPrime = false;
                    break;
                }
            }

            if (isPrime) System.out.print(i + " ");
        }
    }
}
```

### ✅ **해설**
- 소수는 **1과 자기 자신만으로 나누어지는 수**  
- 각 숫자 i에 대해 2부터 i-1까지 나누어보며 소수 여부 판단  
- 나누어 떨어지면 소수가 아님 → `break`

# ✅ 2. 자리수 합 구하기

### ✅ **정답 코드**
```java
import java.util.Scanner;

public class Q2 {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        int num = sc.nextInt();
        int sum = 0;

        while (num > 0) {
            sum += num % 10;
            num /= 10;
        }

        System.out.println(sum);
    }
}
```

### ✅ **해설**
- `% 10` → 마지막 자리 추출  
- `/ 10` → 마지막 자리 제거  
- 반복하며 모든 자리수를 더함

# ✅ 5. 1부터 N까지 완전수 출력

### ✅ **정답 코드**
```java
import java.util.Scanner;

public class Q5 {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        int N = sc.nextInt();

        for (int i = 1; i <= N; i++) {
            int sum = 0;

            for (int j = 1; j < i; j++) {
                if (i % j == 0) sum += j;
            }

            if (sum == i) System.out.print(i + " ");
        }
    }
}
```

### ✅ **해설**
- 완전수 = 자기 자신을 제외한 약수의 합이 자기 자신  
- 예: 6 → 1+2+3 = 6

# ✅ 6. 숫자 뒤집기 & 회문 판별

### ✅ **정답 코드**
```java
import java.util.Scanner;

public class Q6 {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        int num = sc.nextInt();
        int original = num;
        int rev = 0;

        while (num > 0) {
            rev = rev * 10 + num % 10;
            num /= 10;
        }

        if (rev == original) System.out.println("회문수");
        else System.out.println("회문 아님");
    }
}
```

### ✅ **해설**
- 숫자를 뒤집는 과정은 자리수 합과 동일한 방식  
- 뒤집은 값과 원래 값이 같으면 회문수

# ✅ 8. 중앙 정렬 삼각형 출력

### ✅ **정답 코드**
```java
import java.util.Scanner;

public class Q8 {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        int N = sc.nextInt();

        for (int i = 1; i <= N; i++) {
            for (int s = 1; s <= N - i; s++) System.out.print(" ");
            for (int star = 1; star <= 2 * i - 1; star++) System.out.print("*");
            System.out.println();
        }
    }
}
```

### ✅ **해설**
- 공백 출력 → `N - i`  
- 별 출력 → `2*i - 1`  
- 중앙 정렬의 핵심은 **공백 먼저, 별 나중**

# ✅ 9. 숫자 맞히기 게임

### ✅ **정답 코드**
```java
import java.util.*;

public class Q9 {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        Random r = new Random();

        int answer = r.nextInt(50) + 1;
        int guess = 0;

        while (guess != answer) {
            guess = sc.nextInt();

            if (guess > answer) System.out.println("Down");
            else if (guess < answer) System.out.println("Up");
            else System.out.println("정답!");
        }
    }
}
```

### ✅ **해설**
- 난수 생성: `1~50`  
- 입력값이 크면 Down, 작으면 Up  
- 맞으면 반복 종료

# ✅ 10. 최대공약수(GCD)

### ✅ **정답 코드**
```java
import java.util.Scanner;

public class Q10 {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        int a = sc.nextInt();
        int b = sc.nextInt();

        int gcd = 1;

        for (int i = 1; i <= Math.min(a, b); i++) {
            if (a % i == 0 && b % i == 0) gcd = i;
        }

        System.out.println("GCD = " + gcd);
    }
}
```

### ✅ **해설**
- 1부터 두 수 중 작은 값까지 반복  
- 둘 다 나누어 떨어지는 가장 큰 수가 GCD  
- 가장 기본적인 반복문 방식
