# 엑셀(Excel)의 데이터 구조
 - 셀(Cell) →
 - 시트(Sheet) →
 -  문서(Workbook)

---

## 🟢 엑셀 데이터 구조

### 1. 셀(Cell)
- **정의**: 엑셀에서 가장 작은 데이터 단위. 행(Row)과 열(Column)의 교차점.
- **특징**:
  - 하나의 값(숫자, 문자, 수식 등)을 저장.
  - 주소로 식별 (예: `A1`, `B3`).
  - 수식 입력 가능 (`=SUM(A1:A5)`).

---

### 2. 시트(Sheet)
- **정의**: 셀들이 모여 있는 하나의 표 형태 공간.
- **특징**:
  - 여러 행과 열로 구성.
  - 하나의 문서 안에 여러 시트를 가질 수 있음.
  - 시트 이름으로 구분 (예: "매출현황", "재고관리").

---

### 3. 문서(Workbook)
- **정의**: 여러 시트가 모여 있는 하나의 엑셀 파일.
- **특징**:
  - 확장자: `.xlsx`, `.xls`.
  - 문서 단위로 저장/공유.
  - 시트 간 데이터 참조 가능 (`=Sheet2!A1`).

---

## ✨ 구조 요약

| 계층 | 예시 | 특징 |
|------|------|------|
| 셀(Cell) | A1, B3 | 가장 작은 단위, 값/수식 저장 |
| 시트(Sheet) | "매출현황" 시트 | 셀들의 집합, 표 형태 |
| 문서(Workbook) | sales.xlsx | 여러 시트 포함, 파일 단위 |

---

## 🧩 효율적인 사용 방법
- **셀**: 데이터 입력 시 일관된 형식 유지 (숫자/문자 구분).
- **시트**: 목적별로 분리 (예: "매출", "비용", "재고").
- **문서**: 하나의 프로젝트/업무 단위로 관리.

---

## 🧩 간단한 예제

### 예제: 매출 관리 문서
- **문서 이름**: `SalesReport.xlsx`
- **시트 구성**:
  - `월별매출` 시트: 각 월 매출 기록
  - `제품별매출` 시트: 제품별 매출 기록
  - `요약` 시트: 전체 합계 및 분석

**월별매출 시트 예시**
| 월   | 매출액   |
|------|----------|
| 1월  | 1,000,000 |
| 2월  | 1,200,000 |
| 3월  | 900,000   |

**요약 시트 예시**
```excel
=SUM(월별매출!B2:B4)   // 총 매출 합계
=AVERAGE(월별매출!B2:B4) // 평균 매출
```

👉 이렇게 하면 **셀 단위 데이터 → 시트별 관리 → 문서 전체 분석** 구조로 효율적인 관리가 가능합니다.

---
