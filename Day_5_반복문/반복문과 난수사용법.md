# 자바 반복문과 난수법
> 자바의 반복문은 아래와 같이 3가지 유형으로 기술할 수 있으며, 각각 상황에 따라 적절하게 선택하여 기술하면 된다.
- for
- while
- do~while

---

## 🔁 1. `for` 반복문
### 📌 문법
```java
for (초기식; 조건식; 증감식) {
    // 반복 실행할 코드
}
```
- **초기식**: 반복 변수 초기화
- **조건식**: 반복을 계속할지 여부 판단
- **증감식**: 반복 변수 변경

### ⚡ 특징
- 반복 횟수가 **명확히 정해져 있을 때** 사용
- 배열, 리스트 순회에 자주 활용

### 🧩 대표 예제
1. **1부터 10까지 합 구하기**
```java
int sum = 0;
for (int i = 1; i <= 10; i++) {
    sum += i;
}
System.out.println("합: " + sum);
```

2. **구구단 출력**
```java
for (int i = 2; i <= 9; i++) {
    for (int j = 1; j <= 9; j++) {
        System.out.println(i + " x " + j + " = " + (i*j));
    }
}
```

3. **배열 순회**
```java
int[] arr = {10, 20, 30, 40};
for (int i = 0; i < arr.length; i++) {
    System.out.println(arr[i]);
}
```

## 🔁 2. `while` 반복문
### 📌 문법
```java
while (조건식) {
    // 반복 실행할 코드
}
```
- 조건식이 **true일 동안** 반복
- 반복 횟수가 **불명확할 때** 적합

### ⚡ 특징
- 조건을 먼저 검사 → 조건이 false면 한 번도 실행되지 않을 수 있음
- 입력값 검증, 무한 루프 등에 자주 사용

### 🧩 대표 예제
1. **사용자 입력 받기 (0 입력 시 종료)**
```java
Scanner sc = new Scanner(System.in);
int num;
while ((num = sc.nextInt()) != 0) {
    System.out.println("입력값: " + num);
}
```

2. **1부터 100까지 짝수 출력**
```java
int i = 1;
while (i <= 100) {
    if (i % 2 == 0) {
        System.out.println(i);
    }
    i++;
}
```

3. **랜덤 숫자 맞추기 게임**
```java
int target = (int)(Math.random() * 10) + 1;
int guess = -1;
Scanner sc = new Scanner(System.in);

while (guess != target) {
    System.out.print("숫자를 맞춰보세요: ");
    guess = sc.nextInt();
}
System.out.println("정답!");
```

## 🔁 3. `do~while` 반복문
### 📌 문법
```java
do {
    // 반복 실행할 코드
} while (조건식);
```
- **조건을 나중에 검사** → 최소 1회는 반드시 실행됨

### ⚡ 특징
- 사용자 입력을 최소 한 번은 받아야 하는 경우 유용
- 메뉴 선택, 재시도 로직 등에 자주 활용

### 🧩 대표 예제
1. **메뉴 선택 반복**
```java
int choice;
Scanner sc = new Scanner(System.in);
do {
    System.out.println("1. 시작 | 2. 종료");
    choice = sc.nextInt();
} while (choice != 2);
System.out.println("프로그램 종료");
```

2. **비밀번호 확인**
```java
String password;
Scanner sc = new Scanner(System.in);
do {
    System.out.print("비밀번호 입력: ");
    password = sc.nextLine();
} while (!password.equals("1234"));
System.out.println("접속 성공");
```

3. **1부터 10까지 출력**
```java
int i = 1;
do {
    System.out.println(i);
    i++;
} while (i <= 10);
```

|--------------|-------------------|---------------|-----------|

## 🎲 자바 난수 생성 방법 정리

### 1. `Math.random()`
- 0.0 이상 1.0 미만의 **double** 값을 반환.
- 범위를 조정하려면 곱셈과 덧셈을 활용.
```java
int n = (int)(Math.random() * 10); // 0~9 사이 정수
```

### 2. `Random` 클래스 (`java.util.Random`)
- 다양한 타입의 난수 제공 (`nextInt()`, `nextDouble()`, `nextBoolean()` 등).
```java
Random rand = new Random();
int n = rand.nextInt(10); // 0~9 사이 정수
```

### 3. `ThreadLocalRandom` (멀티스레드 환경에 적합)
```java
int n = ThreadLocalRandom.current().nextInt(1, 101); // 1~100 사이 정수
```

### 4. `SecureRandom` (보안용 난수)
- 암호학적으로 안전한 난수 필요 시 사용.
```java
SecureRandom sr = new SecureRandom();
int n = sr.nextInt(100); // 0~99 사이 정수
```

## 🧩 간단한 사용 예제 5개

### 1. 주사위 굴리기 (Math.random)
```java
int dice = (int)(Math.random() * 6) + 1;
System.out.println("주사위: " + dice);
```

### 2. 랜덤 비밀번호 생성 (Random)
```java
Random rand = new Random();
String chars = "ABCDEFGHIJKLMNOPQRSTUVWXYZ0123456789";
StringBuilder pw = new StringBuilder();
for (int i = 0; i < 8; i++) {
    pw.append(chars.charAt(rand.nextInt(chars.length())));
}
System.out.println("비밀번호: " + pw);
```

### 3. 배열 섞기 (배열 활용 ①)
```java
int[] arr = {1, 2, 3, 4, 5};
Random rand = new Random();
for (int i = 0; i < arr.length; i++) {
    int j = rand.nextInt(arr.length);
    // swap
    int temp = arr[i];
    arr[i] = arr[j];
    arr[j] = temp;
}
System.out.println(Arrays.toString(arr));
```

### 4. 배열에서 랜덤 추출 (배열 활용 ②)
```java
String[] fruits = {"사과", "바나나", "포도", "딸기"};
Random rand = new Random();
String pick = fruits[rand.nextInt(fruits.length)];
System.out.println("선택된 과일: " + pick);
```

### 5. 랜덤 로또 번호 생성 (ThreadLocalRandom)
```java
Set<Integer> lotto = new HashSet<>();
while (lotto.size() < 6) {
    lotto.add(ThreadLocalRandom.current().nextInt(1, 46));
}
System.out.println("로또 번호: " + lotto);
```

## 🎲 난수 범위 공식

자바에서 난수를 특정 범위 `[start, end]`로 얻는 공식은 다음과 같습니다:

### 1. `Math.random()` 이용
```java
int rand = (int)(Math.random() * (end - start + 1)) + start;
```
- `Math.random()` → 0.0 이상 1.0 미만의 double 값 반환
- `(end - start + 1)` → 범위 크기
- `+ start` → 시작값 보정

### 2. `Random` 클래스 이용
```java
Random r = new Random();
int rand = r.nextInt(end - start + 1) + start;
```
- `nextInt(n)` → 0 이상 n 미만의 정수 반환
- `+ start` → 시작값 보정

## 🧩 사용자 입력 기반 난수 생성 프로그램

```java
import java.util.Random;
import java.util.Scanner;

public class RandomRangeExample {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);

        // 사용자 입력 받기
        System.out.print("시작값을 입력하세요: ");
        int start = sc.nextInt();

        System.out.print("종료값을 입력하세요: ");
        int end = sc.nextInt();

        // Random 객체 생성
        Random rand = new Random();

        // 난수 생성
        int randomNumber = rand.nextInt(end - start + 1) + start;

        // 결과 출력
        System.out.println("생성된 난수: " + randomNumber);
    }
}
```
