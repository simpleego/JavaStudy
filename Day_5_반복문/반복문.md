# 자바 반복문 
> 자바의 반복문은 아래와 같이 3가지 유형으로 기술할 수 있으며, 각각 상황에 따라 적절하게 선택하여 기술하면 된다.
- for
- while
- do~while

---

## 🔁 1. `for` 반복문
### 📌 문법
```java
for (초기식; 조건식; 증감식) {
    // 반복 실행할 코드
}
```
- **초기식**: 반복 변수 초기화
- **조건식**: 반복을 계속할지 여부 판단
- **증감식**: 반복 변수 변경

### ⚡ 특징
- 반복 횟수가 **명확히 정해져 있을 때** 사용
- 배열, 리스트 순회에 자주 활용

### 🧩 대표 예제
1. **1부터 10까지 합 구하기**
```java
int sum = 0;
for (int i = 1; i <= 10; i++) {
    sum += i;
}
System.out.println("합: " + sum);
```

2. **구구단 출력**
```java
for (int i = 2; i <= 9; i++) {
    for (int j = 1; j <= 9; j++) {
        System.out.println(i + " x " + j + " = " + (i*j));
    }
}
```

3. **배열 순회**
```java
int[] arr = {10, 20, 30, 40};
for (int i = 0; i < arr.length; i++) {
    System.out.println(arr[i]);
}
```

## 🔁 2. `while` 반복문
### 📌 문법
```java
while (조건식) {
    // 반복 실행할 코드
}
```
- 조건식이 **true일 동안** 반복
- 반복 횟수가 **불명확할 때** 적합

### ⚡ 특징
- 조건을 먼저 검사 → 조건이 false면 한 번도 실행되지 않을 수 있음
- 입력값 검증, 무한 루프 등에 자주 사용

### 🧩 대표 예제
1. **사용자 입력 받기 (0 입력 시 종료)**
```java
Scanner sc = new Scanner(System.in);
int num;
while ((num = sc.nextInt()) != 0) {
    System.out.println("입력값: " + num);
}
```

2. **1부터 100까지 짝수 출력**
```java
int i = 1;
while (i <= 100) {
    if (i % 2 == 0) {
        System.out.println(i);
    }
    i++;
}
```

3. **랜덤 숫자 맞추기 게임**
```java
int target = (int)(Math.random() * 10) + 1;
int guess = -1;
Scanner sc = new Scanner(System.in);

while (guess != target) {
    System.out.print("숫자를 맞춰보세요: ");
    guess = sc.nextInt();
}
System.out.println("정답!");
```

## 🔁 3. `do~while` 반복문
### 📌 문법
```java
do {
    // 반복 실행할 코드
} while (조건식);
```
- **조건을 나중에 검사** → 최소 1회는 반드시 실행됨

### ⚡ 특징
- 사용자 입력을 최소 한 번은 받아야 하는 경우 유용
- 메뉴 선택, 재시도 로직 등에 자주 활용

### 🧩 대표 예제
1. **메뉴 선택 반복**
```java
int choice;
Scanner sc = new Scanner(System.in);
do {
    System.out.println("1. 시작 | 2. 종료");
    choice = sc.nextInt();
} while (choice != 2);
System.out.println("프로그램 종료");
```

2. **비밀번호 확인**
```java
String password;
Scanner sc = new Scanner(System.in);
do {
    System.out.print("비밀번호 입력: ");
    password = sc.nextLine();
} while (!password.equals("1234"));
System.out.println("접속 성공");
```

3. **1부터 10까지 출력**
```java
int i = 1;
do {
    System.out.println(i);
    i++;
} while (i <= 10);
```

## 📊 요약 비교

| 반복문       | 실행 조건 검사 시점 | 최소 실행 횟수 | 대표 활용 |
|--------------|-------------------|---------------|-----------|
| **for**      | 반복 전 검사       | 0회 이상      | 정해진 횟수 반복, 배열 순회 |
| **while**    | 반복 전 검사       | 0회 이상      | 조건 기반 반복, 무한 루프 |
| **do~while** | 반복 후 검사       | 1회 이상      | 메뉴 선택, 입력 검증 |

---

👉 이렇게 정리하면, **for는 횟수 기반, while은 조건 기반, do~while은 최소 1회 실행 보장**이라는 차이가 확실히 드러납니다.
