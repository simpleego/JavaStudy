# 자바 연산자와 형변환
-----

## 1\. Java 연산자 (Operators)

Java 연산자는 변수나 값에 대해 특정 작업을 수행하는 데 사용되는 기호입니다.

### 1.1. 산술 연산자 (Arithmetic Operators)

수학적 계산에 사용됩니다.

| 연산자 | 이름 | 설명 |
| :---: | :---: | :--- |
| `+` | 덧셈 | 두 값을 더합니다. |
| `-` | 뺄셈 | 오른쪽 값에서 왼쪽 값을 먽 뺍니다. |
| `*` | 곱셈 | 두 값을 곱합니다. |
| `/` | 나눗셈 | 왼쪽 값을 오른쪽 값으로 나눕니다. (정수/정수 연산 시 **몫**을 구함) |
| `%` | 나머지 | 왼쪽 값을 오른쪽 값으로 나눈 **나머지**를 구합니다. |

```java
public class ArithmeticExample {
    public static void main(String[] args) {
        int a = 10;
        int b = 3;

        System.out.println("a + b = " + (a + b)); // 13
        System.out.println("a - b = " + (a - b)); // 7
        System.out.println("a * b = " + (a * b)); // 30
        System.out.println("a / b (정수 나눗셈) = " + (a / b)); // 3 (몫)
        System.out.println("a % b (나머지 연산) = " + (a % b)); // 1 (나머지)

        double c = 10.0;
        double d = 3.0;
        System.out.println("c / d (실수 나눗셈) = " + (c / d)); // 3.3333...
    }
}
```

### 1.2. 대입 연산자 (Assignment Operators)

변수에 값을 할당하는 데 사용됩니다.

| 연산자 | 예시 | 설명 | 동일 표현 |
| :---: | :---: | :--- | :---: |
| `=` | `a = 5` | `a`에 5를 할당 | |
| `+=` | `a += 2` | `a = a + 2`와 같음 | `a = a + 2` |
| `-=` | `a -= 2` | `a = a - 2`와 같음 | `a = a - 2` |
| `*=` | `a *= 2` | `a = a * 2`와 같음 | `a = a * 2` |
| `/=` | `a /= 2` | `a = a / 2`와 같음 | `a = a / 2` |
| `%=` | `a %= 2` | `a = a % 2`와 같음 | `a = a % 2` |

```java
public class AssignmentExample {
    public static void main(String[] args) {
        int x = 10;
        x += 5; // x = x + 5; (15)
        System.out.println("x += 5 결과: " + x);

        x /= 3; // x = x / 3; (5)
        System.out.println("x /= 3 결과: " + x);
    }
}
```

### 1.3. 비교 연산자 (Comparison Operators)

두 값을 비교하고 논리 값(`true` 또는 `false`)을 반환합니다.

| 연산자 | 이름 | 설명 |
| :---: | :---: | :--- |
| `==` | 같다 | 두 값이 같으면 `true` |
| `!=` | 같지 않다 | 두 값이 다르면 `true` |
| `>` | 크다 | 왼쪽 값이 오른쪽 값보다 크면 `true` |
| `<` | 작다 | 왼쪽 값이 오른쪽 값보다 작으면 `true` |
| `>=` | 크거나 같다 | 왼쪽 값이 오른쪽 값보다 크거나 같으면 `true` |
| `<=` | 작거나 같다 | 왼쪽 값이 오른쪽 값보다 작거나 같으면 `true` |

```java
public class ComparisonExample {
    public static void main(String[] args) {
        int i = 5;
        int j = 10;

        System.out.println("i == j: " + (i == j)); // false
        System.out.println("i != j: " + (i != j)); // true
        System.out.println("i < j: " + (i < j));   // true
    }
}
```

### 1.4. 논리 연산자 (Logical Operators)

논리 값(`boolean`)을 사용하여 논리 연산을 수행합니다.

| 연산자 | 이름 | 설명 |
| :---: | :---: | :--- |
| `&&` | 논리 AND | 두 조건이 모두 `true`일 때만 `true` |
| `||` | 논리 OR | 두 조건 중 하나라도 `true`이면 `true` |
| `!` | 논리 NOT | 논리 값을 반전 (`true`는 `false`로, `false`는 `true`로) |

```java
public class LogicalExample {
    public static void main(String[] args) {
        boolean isSunny = true;
        boolean isWeekend = false;

        // isSunny 이고 isWeekend 이면
        System.out.println("AND (&&): " + (isSunny && isWeekend)); // false

        // isSunny 이거나 isWeekend 이면
        System.out.println("OR (||): " + (isSunny || isWeekend));   // true

        // isSunny의 반대
        System.out.println("NOT (!): " + (!isSunny));              // false
    }
}
```

### 1.5. 증감 연산자 (Increment/Decrement Operators)

변수 값을 1씩 증가시키거나 감소시킵니다.

| 연산자 | 이름 | 설명 |
| :---: | :---: | :--- |
| `++` | 증가 | 변수 값을 1 증가시킵니다. |
| `--` | 감소 | 변수 값을 1 감소시킵니다. |

> **전위(Prefix)**: `++a`, `--a` (먼저 1 증가/감소 후 연산)
> **후위(Postfix)**: `a++`, `a--` (연산이 끝난 후 1 증가/감소)

```java
public class IncrementDecrementExample {
    public static void main(String[] args) {
        int num1 = 5;
        int result1 = ++num1; // num1을 6으로 만들고, result1에 6 대입

        System.out.println("전위(++num1) - num1: " + num1 + ", result1: " + result1); // 6, 6

        int num2 = 5;
        int result2 = num2++; // result2에 5를 먼저 대입하고, num2를 6으로 만듦

        System.out.println("후위(num2++) - num2: " + num2 + ", result2: " + result2); // 6, 5
    }
}
```

## 2\. Java 형 변환 (Type Casting)

형 변환은 하나의 데이터 타입을 다른 데이터 타입으로 바꾸는 것을 의미합니다.

### 2.1. 자동 형 변환 (묵시적 형 변환, Implicit Casting)

**작은 크기의 타입**이 **큰 크기의 타입**으로 변환될 때 자동으로 발생합니다. 데이터 손실이 없습니다.

  * **변환 순서:** `byte` -\> `short` -\> `int` -\> `long` -\> `float` -\> `double`

<!-- end list -->

```java
public class ImplicitCastingExample {
    public static void main(String[] args) {
        int intValue = 100;
        double doubleValue = intValue; // int -> double (자동 변환)

        System.out.println("intValue: " + intValue);     // 100
        System.out.println("doubleValue: " + doubleValue); // 100.0
    }
}
```

### 2.2. 강제 형 변환 (명시적 형 변환, Explicit Casting)

**큰 크기의 타입**을 **작은 크기의 타입**으로 변환할 때 사용자가 직접 \*\*(타입)\*\*을 명시해 주어야 합니다. 데이터 손실(값 잘림)이 발생할 수 있습니다.

```java
public class ExplicitCastingExample {
    public static void main(String[] args) {
        double doubleValue = 10.75;
        int intValue = (int) doubleValue; // double -> int (강제 변환, 소수점 이하 버림)

        System.out.println("doubleValue: " + doubleValue); // 10.75
        System.out.println("intValue: " + intValue);       // 10

        // 큰 값에서 작은 타입으로 변환 시 데이터 손실 발생 가능
        int largeInt = 130;
        byte byteValue = (byte) largeInt; // int -> byte (byte의 범위는 -128~127)

        System.out.println("largeInt: " + largeInt);     // 130
        System.out.println("byteValue: " + byteValue);   // -126 (데이터 손실)
    }
}
```

## 3\. 나머지 연산자 (`%`) 응용 예제 3가지
-----

### 응용 예제 1: 짝수/홀수 판별 결과를 논리 값으로 변환

```java
public class ModuloApplication1_NoIf {
    public static void main(String[] args) {
        int number = 15;

        // 짝수 판별: number % 2 == 0 은 짝수일 때만 true (나머지가 0)
        boolean isEven = (number % 2 == 0);
        
        // 홀수 판별: number % 2 != 0 은 홀수일 때만 true (나머지가 1)
        boolean isOdd = (number % 2 != 0);

        System.out.println(number + "은(는) 짝수입니까? " + isEven); // false
        System.out.println(number + "은(는) 홀수입니까? " + isOdd);   // true
        
        int number2 = 20;
        System.out.println(number2 + "은(는) 짝수입니까? " + (number2 % 2 == 0)); // true
    }
}
```

### 응용 예제 2: 특정 주기/간격마다 0으로 초기화 (카운터 순환)

`%` 연산자는 특정 범위 내에서 숫자를 순환(Wrap around)시키는 데 매우 유용하며, 배열 인덱스를 다룰 때 자주 사용됩니다. 여기서는 1부터 5까지 카운트한 후 다시 1로 돌아가도록 만듭니다.

```java
public class ModuloApplication2_NoIf {
    public static void main(String[] args) {
        int maxCount = 5;

        // 카운터: 0, 1, 2, 3, 4, 5, 6, 7, ...
        int counter1 = 3; 
        int counter2 = 5; 
        int counter3 = 8;
        
        // 순환 카운터: (현재 카운터 % 최대 주기)
        // 주의: 0부터 시작하는 순환을 만들 때는 (값 % 주기)를 사용하고,
        // 1부터 시작하는 순환을 만들 때는 (값 - 1) % 주기 + 1 을 사용합니다.

        // (3 % 5) = 3
        int cycleValue1 = counter1 % maxCount; 
        
        // (5 % 5) = 0
        int cycleValue2 = counter2 % maxCount; 
        
        // (8 % 5) = 3
        int cycleValue3 = counter3 % maxCount; 

        System.out.println(counter1 + "번째의 순환 값 (0부터): " + cycleValue1); // 3
        System.out.println(counter2 + "번째의 순환 값 (0부터): " + cycleValue2); // 0 (5가 되면 리셋)
        System.out.println(counter3 + "번째의 순환 값 (0부터): " + cycleValue3); // 3
    }
}
```

### 응용 예제 3: 자릿수 추출 및 시간 계산 (가장 흔한 응용)

정수에서 가장 오른쪽 자릿수(1의 자리)를 추출하거나, 큰 단위의 시간을 작은 단위로 분해할 때 사용됩니다. 이는 `if`문 없이 순수하게 연산만으로 해결되는 가장 대표적인 활용입니다.

```java
public class ModuloApplication3_NoIf {
    public static void main(String[] args) {
        int value = 4567;

        // 1. 1의 자리 추출
        // 10으로 나눈 나머지는 항상 1의 자리 숫자입니다.
        int onesDigit = value % 10; 
        System.out.println(value + "의 1의 자리는: " + onesDigit); // 7

        // 2. 시/분/초 변환
        int totalSeconds = 7500; // 7500초
        
        // 60으로 나눈 나머지 = 남은 초 (0~59)
        int seconds = totalSeconds % 60; 
        
        // 60으로 나눈 몫 = 전체 분 (125분)
        int totalMinutes = totalSeconds / 60; 

        // 전체 분을 60으로 나눈 나머지 = 남은 분 (0~59)
        int minutes = totalMinutes % 60; 
        
        // 전체 분을 60으로 나눈 몫 = 시간 (2시간)
        int hours = totalMinutes / 60;   

        System.out.println(totalSeconds + "초는 " + hours + "시간 " + minutes + "분 " + seconds + "초 입니다."); 
        // 7500초는 2시간 5분 0초 입니다.
    }
}
```

### 응용 예제 1: 짝수/홀수 판별 (Even/Odd Check)

어떤 수를 2로 나누었을 때 나머지가 0이면 **짝수**, 1이면 **홀수**입니다.
```java
public class ModuloApplication3 {
    public static void main(String[] args) {
        int value = 456;

        // 10으로 나눈 나머지: 1의 자리 추출
        int onesDigit = value % 10; 
        System.out.println(value + "의 1의 자리는: " + onesDigit); // 6

        // 시간 계산 응용: 초를 시/분/초로 변환
        int totalSeconds = 7500; // 7500초
        
        int seconds = totalSeconds % 60; // 60으로 나눈 나머지 = 남은 초 (0~59)
        int totalMinutes = totalSeconds / 60; // 60으로 나눈 몫 = 전체 분

        int minutes = totalMinutes % 60; // 60으로 나눈 나머지 = 남은 분 (0~59)
        int hours = totalMinutes / 60;   // 60으로 나눈 몫 = 시간

        System.out.println(totalSeconds + "초는 " + hours + "시간 " + minutes + "분 " + seconds + "초 입니다."); 
        // 7500초는 2시간 5분 0초 입니다. (7500/60 = 125분, 125/60 = 2시간 5분)
    }
}
```
